Printer-Friendly:
Using the Stack Exchange Network API produces an essential printable view of a given question id with:
* All the answers
* All the comments

The easiest way to use this service is with:
* StackPrinter GreaseMonkey script
* Bookmarklet

MySE:
Retrieves and lets you print all your Stack Exchange stuff

Favorites:
Retrieves and lets you print all the favorites users' questions

Top Voted:
Retrieves and lets you print the top voted questions by tags

Top Printed:
Top voted questions list

Deleted:
Deleted questions list
